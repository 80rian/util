# Instagram Reels Cover Image Generator

## Overview

인스타그램 릴스에 올릴 커버 이미지를 생성하는 로컬 웹 도구. 배경 이미지 위에 타이틀 텍스트를 중앙에 배치하고, 프리셋 스타일을 적용한 뒤 PNG로 다운로드한다.

## Architecture

**Single HTML file** — 브라우저에서 파일을 열면 바로 동작. 서버 불필요. HTML, CSS, JS가 하나의 파일에 포함된다. 향후 필요시 React로 마이그레이션 가능.

핵심 기술: Canvas API로 이미지 합성 및 렌더링.

## Workflow

1. 이미지 업로드 (드래그&드롭 또는 파일 선택)
2. 타이틀 텍스트 입력
3. 프리셋 스타일 선택 (자동 색상 추천 표시)
4. 실시간 미리보기 확인 (릴스 9:16 + 피드 1:1 크롭)
5. PNG 다운로드 (1080x1920)

## Output

- **생성 이미지**: 1080x1920px (9:16) PNG
- **미리보기 2종**:
  - 릴스 미리보기: 9:16 원본
  - 피드 미리보기: 9:16 이미지의 중앙을 1:1로 크롭한 결과
- 타이틀은 양쪽 미리보기 모두에서 잘리지 않도록 **중앙 배치**

## Presets (7종)

각 프리셋은 JS 객체로 정의. 텍스트 색상, 그림자, 배경 효과 등을 포함한다.

| # | 이름 | 설명 | 용도 |
|---|------|------|------|
| 1 | 클린 화이트 | 흰색 볼드 + 소프트 그림자 (`text-shadow: 0 2px 12px rgba(0,0,0,0.6)`) | 어두운 배경. 가장 범용적 |
| 2 | 하드 섀도우 | 흰색 볼드 + 오프셋 하드 그림자 (`3px 3px 0px rgba(0,0,0,0.8)`) | 레트로/강렬한 느낌 |
| 3 | 다크 모던 | 검정 텍스트, 그림자 없음 | 밝은 배경 |
| 4 | 솔리드 라벨 | 반투명 검정 박스(`rgba(0,0,0,0.75)`) + 흰색 텍스트 | 최고 가독성, 어떤 배경이든 |
| 5 | 프로스트 글래스 | 텍스트 영역 이미지를 Canvas로 크롭→블러→반투명 오버레이 + 흰색 텍스트 | 프리미엄/모던 |
| 6 | 하이라이트 마커 | 줄별 형광색 배경 스트라이프 + 어두운 텍스트 | 캐주얼/친근한 느낌 |
| 7 | 그라디언트 스크림 | 하단 30-50% 그라디언트 오버레이(투명→검정) + 흰색 텍스트 | 시네마틱/자연스러운 느낌 |

프리셋 데이터 구조 예시:
```js
const PRESETS = {
  cleanWhite: {
    name: '클린 화이트',
    textColor: '#ffffff',
    fontWeight: 800,
    shadow: { blur: 12, offsetX: 0, offsetY: 2, color: 'rgba(0,0,0,0.6)' },
    background: null,
  },
  // ...
};
```

## Auto Color Recommendation

배경 이미지의 텍스트 영역(중앙부)의 평균 밝기를 분석하여 적절한 프리셋을 추천한다.

- Canvas에서 중앙 영역 픽셀 데이터 추출
- 평균 luminance 계산 (`0.299*R + 0.587*G + 0.114*B`)
- 밝으면 어두운 텍스트 프리셋 추천, 어두우면 밝은 텍스트 프리셋 추천
- 추천은 UI에 표시만 하고, 사용자가 직접 선택할 수 있음

## Font Support

- **기본**: Google Fonts 웹폰트 (Noto Sans KR, Pretendard 등)
- **로컬 폰트**: 사용자가 .ttf/.otf 파일을 업로드하면 `FontFace` API로 로드하여 사용
- 폰트 선택 드롭다운에서 웹폰트와 로컬 폰트를 함께 표시
